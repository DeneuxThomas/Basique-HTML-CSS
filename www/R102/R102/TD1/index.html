V.2 Les balises de texte

  1. Expliquez ce qu'est une URL et une URI, en étayant vos explications par des extraits de Wikipédia (balises blockquote ou q et balise cite).

    Une URL Uniform Resource Locator ou en français localisateur uniforme de ressource ou plus simplement adresse web est une adresse sur le web qui permet de préciser son emplacement sur le WWW (World Wide Web). Elle s'écrit sous forme de chaîne de caractère et suit un certain protocole internet (le http ou https) puis un une adresse IP, un chemin d'accès...

    Une URI Uniform Resource Identifier ou en français Uniform identifiant uniforme de ressource est une ressource sur un réseau. Elle peut-être physique ou abstraite. Une URI respecte une norme d'internet par exemple la RFC2396

V.3 Insertion d'une image

  1. Qu'est-ce qu'un format d'image matriciel ? Vectoriel ? Avec perte/sans perte ?

    Une image matricielle est une image numérique formée d'une multitude de points de couleur (pixels), et représentée sur une grille dotée de deux axes, X et Y alors que une image vectorielle est faite de formes (polygones, lignes, ellipses) possédant divers paramètres tels que hauteur, longueur, rayon, couleur, etc. Elle est créée sur un ordinateur à partir de formules mathématiqes

  2. Quelle sont les différences entre les formats PNG, GIF, JPG et SVG ? Et Pour quel usage est conseillé chaque format ?

    Le format PNG est un format d'image qui gère la transparence en fonds. Le format JPEG ou PNG est le plus courant C'est un format compressé qui prends le moins de place possible sur le stockage et donc augmente la vistesse d'affichage. En revanche, c'est deux formats sont connus pour leur compression « avec perte », ce qui signifie que la qualité de l'image diminue à mesure que la taille du fichier diminue. Le GIF est utilisé pour les animations d'images. C'est un format compressé sans perte donc ne se dégrade pas. Elle peut aussi gérer la transparence. Enfin le format SVG est le format de dessin factoriel qui est utilisé pour les création de shéma, diagrammes, carte... C'est un format d'image sans perte

  3. De nouveaux formats d'images ont vu le jour récemment : JPEG2000, WebP, APNG, JPEG-XR, BPG, FLIF, JPEGXL, AVIF... Quels sont leurs points forts/points faibles ? (usages prévus, support, compression, brevets éventuels...)

    En général les nouveaux formats d'images sont des évolutions et une optimisation des précédents formats qui peuvent souvent réduire la taille ou le poids de fichier d'une image, vidéo... ou ajouter d'autre avantage par exemple le Jpeg 2000 et Jpeg XR sont une évolution du JPEG. la taille de fichier est encore plus réduite avec une qualitée similaire et support la transparence. Cependant leur principale point négatif c'est qu'ils ne sont pas compatible avec tout les navigateurs ou sites / applications.

V.4 Créer des liens

  1. En général, vaut-il mieux utiliser "_blank" ou "_self" comme valeur de target (et pourquoi) ? Y a-t-il des exceptions ? Lesquelles et pourquoi ?

    _self affiche la page dans l'onglet actuel (comportement par défaut) et reste sur la même page alors que le _blank permet d'ouvrir le lien dans une autre page ce qui est plus utile si on ne veut pas perdre la précédante page.
